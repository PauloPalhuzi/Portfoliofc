Sobre Mim

Sou Paulo Palhuzi, profissional em transição para a área de Tecnologia da Informação, atualmente cursando o 2º módulo de Desenvolvimento de Sistemas na ETEC Guararapes. Tenho 7 anos de experiência administrativa, forte vivência internacional na Argentina (fluente em espanhol), e hoje atuo como estagiário de TI na Prefeitura Municipal de Guararapes. Participei de um Bootcamp Full Stack de 200 horas com foco prático em C#, .NET, Angular e MySQL, desenvolvendo projetos que consolidaram minha lógica de programação. Tenho interesse especial em suporte técnico e, progressivamente, busco migrar para desenvolvimento de sistemas. Possuo perfil proativo, excelente comunicação, disciplina e facilidade de aprendizado. Estou em busca de oportunidades para aplicar meus conhecimentos, crescer profissionalmente e contribuir com as equipes em que atuar.

Meus Projetos

Projeto Loja e-commerce funcional

Este projeto é uma loja virtual desenvolvida com foco em desempenho, usabilidade e organização de produtos. Utiliza HTML5, CSS3 e JavaScript no front-end para criar uma interface moderna, responsiva e interativa, com carrossel de produtos, filtros e um carrinho de compras dinâmico, onde os usuários podem adicionar ou remover itens com atualização automática do valor total. No back-end, foi utilizado PHP para gerenciar cadastros, login de usuários, sessões e o banco de dados, garantindo uma estrutura funcional para simular a operação real de uma loja. O sistema também inclui um painel administrativo, onde é possível cadastrar, editar e remover produtos de forma prática e segura. Este projeto representa uma aplicação completa de conhecimentos em desenvolvimento web, unindo tecnologias de front-end e back-end em um ambiente integrado e funcional.

Make Glamour

O Glamour Rose é um projeto de loja virtual desenvolvido com o objetivo de simular uma experiência real de e-commerce voltado para o público feminino, com foco em cosméticos, beleza e bem-estar. A proposta do site é unir estética, funcionalidade e desempenho, proporcionando uma navegação fluida e intuitiva para o usuário. A interface foi construída utilizando HTML5, CSS3 e JavaScript, garantindo um visual elegante e moderno, com um carrossel de produtos, categorias organizadas, e um sistema de carrinho de compras dinâmico, onde é possível adicionar e remover itens com atualização automática do valor total. O layout é totalmente responsivo, funcionando perfeitamente em computadores, tablets e celulares.
